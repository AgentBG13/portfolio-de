Projects

Project 1: Sensor Data Pipeline

Designed and implemented a robust real-time streaming data pipeline to ingest high-velocity data from thousands of IoT sensors. The architecture utilized Apache Kafka as the primary message broker, ensuring fault-tolerant, low-latency ingestion. For the storage layer, I leveraged Apache Cassandra to handle the massive volume of time-series data, benefiting from its linear scalability and high write throughput. This project required optimizing data models specifically for time-series access patterns to ensure sub-millisecond retrieval times. I also implemented stream processing using Apache Flink to perform real-time windowed aggregations and anomaly detection before data persistence. This solution significantly reduced the latency of actionable insights, providing the operations team with near-instant visibility into sensor health. The end-to-end system was containerized with Docker and orchestrated using Kubernetes, ensuring seamless deployment and scaling. The outcome was a production-grade system that handles millions of events per day with minimal downtime.

Project 2: Medallion Architecture

Spearheaded the development of a scalable data platform using the Databricks Medallion Architecture (Bronze, Silver, Gold layers) to centralize enterprise data. In the Bronze layer, I implemented Delta Lake to capture raw data streams from diverse source systems, ensuring a permanent, immutable record. The Silver layer involved rigorous data cleaning, deduplication, and schema enforcement to create a refined, trustworthy dataset for consumption. Finally, in the Gold layer, I aggregated and curated business-ready datasets tailored for Power BI dashboards and executive reporting. This project involved optimizing Spark jobs for distributed processing and leveraging Delta Lake's ACID transactions to maintain data integrity across all layers. I implemented automated data quality checks throughout the pipeline, drastically reducing manual data validation efforts. This unified approach transformed fragmented data silos into a coherent, high-performance analytical asset, enabling stakeholders to make data-driven decisions with greater speed and accuracy.

Project 3: Data Migration

Led a complex, large-scale data migration initiative, transitioning core analytical workloads from an on-premise Greenplum database to a modern Databricks cloud environment. The project required meticulous planning to ensure zero-downtime cutover and minimal impact on business operations. I developed custom ETL pipelines using PySpark to map and transform legacy SQL schemas to the modern Delta Lake table formats, handling petabytes of historical data. Challenges included re-engineering thousands of complex legacy stored procedures into Spark-based jobs and ensuring data consistency between the source and destination throughout the transition. I implemented an automated data validation framework that ran parallel comparisons of aggregate reports from both systems to guarantee 100% accuracy post-migration. Additionally, I optimized the new cloud data warehouse for cost and performance, resulting in a 40% reduction in query execution time and significantly lower storage costs. This migration successfully modernized our data infrastructure, laying the foundation for future advanced analytics and machine learning initiatives.
