# Portfolio Website Implementation Plan

## Objective
Build a modern, dark-themed personal portfolio website for Naphat Chapha, a Data Engineer, and host it on GitHub Pages.

## Tech Stack
- HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+)
- Design: Dark theme, JetBrains Mono font
- Hosting: GitHub Pages (Deploy from branch)

## Implementation Steps

### Phase 1: Project Setup
1. Initialize a new local Git repository.
2. Create folder structure: `index.html`, `style.css`, `script.js`, and an `assets/` folder.
3. Add initial boilerplate to `index.html`.

### Phase 2: Design & Development
1. **Global Styles:** Implement dark theme color palette and apply JetBrains Mono font.
2. **Hero Section:** Create section with "Naphat Chapha" and "Data Engineer" title.
3. **Projects Section:** Build a grid layout for 3 data engineering projects with placeholders for descriptions and tech stacks.
4. **Contact Section:** Add contact details (e.g., LinkedIn, GitHub, email).

### Phase 3: Deployment
1. Push the local repository to GitHub.
2. Configure GitHub Pages in repository settings:
   - Source: Deploy from branch.
   - Branch: `main` / `/(root)`.
3. Verify site is live at the GitHub Pages URL.

## Verification & Testing
- Ensure the site is responsive on mobile and desktop.
- Verify all links (e.g., social links) work correctly.
- Confirm site renders correctly in major browsers (Chrome/Firefox/Safari).
